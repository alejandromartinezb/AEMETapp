# Proyecto Fullstack – Backend Node + Frontend React

Este proyecto está compuesto por:

- Backend → `./backend_modular`
- Frontend → `./frontend_react`

---

# Instalación y Ejecución Completa del Proyecto

##  Requisitos Previos

Asegúrate de tener instalado:

- Node.js (versión LTS recomendada)
  - node -v
- npm (incluido con Node.js)
  - npm -v

# Instalación y ejecución del Backend
- cd backend_modular
- npm install
- node server.js

# Instalación y ejecución del Frontend
- cd frontend_react
- npm install
- npm run dev
